#团队代码风格统一


##一、IDE 配置 
在项目根目录创建文件`.editconfig`，在不同IDE之间统一编码风格
[EditConfig官网](https://editorconfig.org/)
示例
``` 
# editorconfig.org

root = true

[*]
charset = utf-8
indent_size = 2
indent_style = space
insert_final_newline = true
trim_trailing_whitespace = true

[*.md]
trim_trailing_whitespace = false

```
