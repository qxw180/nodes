# 各种居中

## Block

水平居中：`margin: 0px auto`

垂直水平居中

```css
/* 已知元素宽高 */
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  margin-top: -3em;
  margin-left: -7em;
  width: 14em;
  height: 6em;
}

.center {
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}

.flex-center {
  display: flex;
  align-items: center;
  justify-content: center;
}
```

## Inline

水平居中：`text-align:center`

垂直居中

- 单行文字：
  - 设置相等的`height`和`line-height`属性
- 多行文字
  - 父元素：`display: table-cell; vertical-align:middle;`
