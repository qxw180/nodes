进程

Process

process对象是一个全局变量，用来获取当前Node.js进程的信息、控制当前Node.js进程；

Process Events

process是一个EventEmitter的实例

uncaughtException：未捕获异常，未处理的JavaScript异常会在标准输出中输出错误信息并退出进程。

Process 属性

  • process.argv：Array，返回通过命令行传递的参数列表，
    • 第一个参数默认为process.execPath
    • 第二个参数默认为执行的JS文件路径
