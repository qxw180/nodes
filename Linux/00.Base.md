#Linux基础

##设置命令别名：`alias`
	
	<!-- 别名检测 -->
	type ll
	<!-- 别名设置 -->
	alias ll='ls -l'
	<!-- 别名删除 -->
	unalias ll

##系统信息相关
1. 查看系统信息：`uname -a`
2. 查看ip信息：`ifconfig`
3. 重启：`reboot`或者`shutdown -r now`


##环境变量

###分类
+ 按照生命周期区分 
    - 永久的：需要修改配置文件，变量永久生效
    - 临时的：使用export命令，在当前终端有效，关闭shell失效
+ 按照作用域区分
    - 系统环境变量：对所有用户有效
    - 用户环境变量：对特定用户有效

###配置
+ 查看环境变量：`echo $PATH`
+ 永久修改：
    * 全局修改：编辑`/etc/profile`文件
    * 局部修改：编辑`~/.bash_profile`文件
+ 临时修改：`export $PATH=变量值`
