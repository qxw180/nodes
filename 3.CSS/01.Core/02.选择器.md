#CSS 选择器

CSS选择器大致可分为以下几类

1. 基本选择器
2. 嵌套选择器(派生选择器)
3. 属性选择器
4. 伪元素、伪类
5. 表单选择器
6. 结构选择器

具体可以参考：`/3.CSS/02.CSS3/01.CSS3-选择器.md`

##选择器的权重和优先级
`内联样式 > 内部样式 > 外部样式`
`!important > id > class > tag`

在嵌套选择器中每个选择器都有自己的权重，权重相加大着优先级高

+ id选择器：	100
+ class选择器：	10
+ 标签选择器：	1

例`div p#large ul.list li`权重`1 + 1+100 + 1+10 + 1 = 114`
