## TODO: DEEP 数据代理 & 数据劫持

通过一个对象对另一个对象中的属性进行操作

```JS
let o1 = {x: 1};
let o2 = {y: 2};

Object.defineProperty(o2, 'x', {
  get() {
    return o1.x;
  },
  set(val) {
    o1.x = val;
  }
})
```

Vue 中的 Model 是通过数据代理到 vue 实例对象
