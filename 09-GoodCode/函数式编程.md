## 纯函数

使用纯函数（给定的输入始终能得到相同的输出），没有外部依赖，优点：

- 结果可可控制，可推测，方便测试
- 无依赖，模块化方便复用
- 无副租用
- 代码可读性强

## immutable 不可变数据(纯数据)

```JS
const obj = {
  name: 'lee'
}
Object.freeze(obj); // 浅层冻结

function deepFreeze(obj) {
  Object.freeze(obj);
  for(let i in obj) {
    if(typeof obj[i] === 'object') {
      deepFreeze(obj[i]);
    }
  }
}
```

## 高阶函数

以函数作为输入或输出的函数，作用：

- 抽象
- 缓存，即闭包
- 惰性执行，防抖、节流

```JS
function once(fn) {
  let done = false;
  return function() {
    if(done) {
      return;
    }
    fu();
    done = true;
  }
}
```

## 柯里化

将多元函数转化为一元函数，讲一个接收多个参数的函数转化为接收一个参数返回一个函数的形式，以便再次调用。

bind 方法除了可以绑定 this 外还可以绑定允许参数，即*柯里化*

```JavaScript
var setStyle = function(size, color){
  this.style.fontSize = size;
  this.style.color = color;
}

var title = document.getElementById('t1');
var setH1Style = setStyle.bind(title,'16px');

setH1Style('red');
```

```JS
function add() {
  const _args = [...arguments];
  function fn() {
    _args.push(...arguments);
    return fn;
  }
  fn.toString = function() {
    return _args.reduce((sum, cur) => sum + cur);
  }
  return fn;
}
console.log(add(1)(2)(3)(4).toString()) // 10
console.log(add(1)(1, 2, 3)(2).toString()) // 9
```
