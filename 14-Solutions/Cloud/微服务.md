# 微服务

> MicroService 是一种软件架构风格，它是以专注于单一责任与功能的小型功能区块 (Small Building Blocks) 为基础，利用模块化的方式组合出复杂的大型应用程序，各功能区块使用与语言无关 (Language-Independent/Language agnostic）的 API 集相互通信。

## 功能及数据库拆分

## 故障处理

1. 监控、预警
   1. 服务器状态监控
   2. 数据库状态监控
   3. 运行数据监控
      1. 异常响应
      2. 耗时异常
      3. 流量异常
2. 日志分析
   1. 链路追踪
   2. 日志分析

## 服务治理

- 服务注册、发现
- 监控检测
- 服务熔断
- 服务限流
